# BookMyStayApp

## Running Procedure

Compile the file HostelStayManager.java and then click the Run button to execute the program.

## Program Flow

The program begins execution.

The JVM calls the main() method.

The application title is shown on the screen.

The application version information is displayed.

The program either proceeds to the next operation or terminates.

## Objective

To develop an effective Hostel Reservation System that helps manage and process hostel bookings smoothly.
